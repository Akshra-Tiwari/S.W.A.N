# S.W.A.N. Protocol

**Smart Water Action Network** - A citizen-powered platform to protect and preserve Bhopal's lakes through real-time reporting, interactive maps, and smart insights.

## About

S.W.A.N. Protocol empowers citizens and government officials to collaborate on lake conservation in Bhopal. Report pollution, track water quality, and drive environmental change through community action.

## Features

- Real-time lake health monitoring with interactive maps
- Citizen-powered issue reporting system
- Analytics dashboard with water quality insights
- Gamification to encourage community participation
- Government and citizen role-based access

## Getting Started

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Run the development server: `pnpm dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Leaflet (Interactive Maps)
- Supabase (Authentication & Database)
